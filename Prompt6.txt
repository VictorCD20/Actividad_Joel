Prompt para Stitch – FestEasy

Te comparto el sistema de diseño para generar una aplicación móvil de FestEasy. De momento, analiza la información y guárdala en tu memoria para la construcción de las interfaces que te voy a solicitar.

La información proporcionada es esencial para definir la arquitectura de diseño con precisión y cumplir con los requisitos de alta fidelidad en un sistema centrado en experiencia de usuario y accesibilidad.

Hemos confirmado tres puntos clave que ahora se integran en el prompt final:

Fidelidad y Tema: Se prioriza un prototipo de Alta Fidelidad, bajo un Tema Oscuro festivo y elegante que evoque celebración y confianza.

Logo: El logo de FestEasy debe ser prominente en el encabezado de la aplicación, para fortalecer la identidad de marca desde la primera interacción.

Componente clave (Reserva rápida): Se utilizará un botón de acción destacado (CTA) con la etiqueta “Reservar Ahora”, posicionado estratégicamente en las pantallas de proveedores, alineado con las mejores prácticas de UI móvil.

Descripción de la UI (en un solo párrafo)

La interfaz de FestEasy está diseñada para que el usuario pueda buscar proveedores, comparar precios y reservar servicios de forma rápida y confiable, todo desde un flujo intuitivo. La pantalla presenta un encabezado con el logo de FestEasy, seguido de un campo de búsqueda con filtros inteligentes (categoría, ubicación, presupuesto). Debajo se muestran tarjetas de proveedores con imagen, nombre, calificación, precio y un botón de acción para reservar. Al final, una barra de navegación inferior permite acceder rápidamente a las secciones principales: Inicio, Búsqueda, Reservas, Comunidad y Perfil.
